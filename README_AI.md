# README_AI — ウェルノット 工程管理

- App ID: `APP-0004`
- 管理責任: アプリ管理責任者
- 最終確認日: 2026-07-05

## 目的と正本

システム名称は「工程管理」とする。従業員から見れば工程管理システムであり、管理者権限では受注・人員・予実・利益まで確認できる判断ツールとして扱う。設計正本は `docs/design`、業務ルールは `src/domain`、データ定義は `src/infrastructure/schema.sql` とする。

## AIが実行できること

読取、設計草案、実装、ローカルテスト、架空データによる帳票確認。外部公開、本番データ操作、権限変更、削除、pushは明示承認を要する。

## 禁止事項

- 顧客・個人・秘密情報をリポジトリへ保存しない
- 金額を従業員向けAPI・HTML・帳票へ返さない
- 完了案件のロックや監査履歴を迂回しない
- Excel原本のレイアウトを推測で確定しない
- 検証を省略して完了報告しない

## 必須検証

`npm run check` が成功し、管理者・従業員の権限境界、完了ロック、納期変更理由、帳票のモノクロ判読性を確認する。

## エスカレーション

実データ、認証方式、本番基盤、現行Excel原本、法定保持期間、バックアップ先が必要な判断はシステム管理責任者へ相談する。
