# 機能仕様書 FEAT-0004

- 対象App ID: APP-0004
- 状態: proposed
- 管理責任: アプリ管理責任者
- 最終確認日: 2026-07-05

## 受入条件

- [ ] 従業員の一覧・詳細・工程APIに金額、予算、原価、集計工数、粗利率、Excel操作が含まれない。
- [ ] 従業員は工程状態と自由記入メモだけを変更できる。
- [ ] 管理者は案件、原価、マスタ、改善メモ、Excel出力、削除、ロック解除を操作できる。
- [ ] 完了案件は完了一覧へ分離され、改善メモ以外はロックされる。
- [ ] 納期変更時に理由なしでは保存できず、変更履歴が残る。
- [ ] 一覧に工番、客先、工事名、納期、状態、現在工程、残工程、工程略称・状態記号が表示される。
- [ ] 図面番号管理ON案件だけが図面番号単位の工程を持てる。
- [ ] 予算項目は任意行を追加でき、材料費・外注費・確定工数から差異と粗利率を自動計算する。
- [ ] A4横工程管理票、A3横受注一覧、A3横ガントを印刷/PDF保存できる。
- [ ] ガントは初期2か月、月送り・横スクロール、案件詳細リンク、モノクロ識別に対応する。
- [ ] Excelは期間指定に応じ、1か月は単月、2〜3か月は月別、4か月以上は期間合算で出力し履歴を残す。

## 例外系

- 権限不足は404相当で機密機能の存在を露出しない。
- 競合更新は更新版番号の不一致として拒否する。
- 帳票範囲外・不正日付・終了日が開始日前の入力を拒否する。
- 削除済み・ロック済み案件への更新を拒否し、管理者の明示的解除を要求する。
